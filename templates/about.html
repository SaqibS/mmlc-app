Integration

Benetech provides three usage and adoption models:
  1. Use Benetech's app - test the tool and process small, one-off requests, at https://mathmlcloud.org.
  2. Interact directly via an API - see API Definition for more information on working with the Benetech-hosted instance of MathML Cloud.
  3. Host your own version of the tool - visit Github to get the code and host the tool on your own, private server, at http://github.com/benetech/mmlc-experiments.
